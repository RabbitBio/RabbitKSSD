PURPOSE

RabbitKSSD is a high-performance genome distance estimation application on modern multi-core architectures.
Specifically, RabbitKSSD leverages load-balanced task partition, fast I/O, efficient intermediate result access, and high-performance data structures to improve the overall efficiency. 
It is implemented in C++ and is distributed with:
